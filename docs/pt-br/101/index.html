FxDart 101

Um curso prático. Cada aula é uma explicação curta com demos ao vivo e um playground — o código compila com o compilador Dart de verdade e roda no seu navegador. Percorra as seções em ordem ou pule para qualquer função abaixo.

Chegou agora? Comece com fx → map → filter → take → toList, e então mergulhe em concurrent.
