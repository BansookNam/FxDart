some

True when at least one element satisfies a predicate — false for an empty iterable.

bool some<A>(bool Function(A a) f, Iterable<A> iterable) Future<bool> someAsync<A>(FutureOr<bool> Function(A a) f, FxAsyncIterable<A> iterable) bool Fx.some(bool Function(T a) f) // chain Future<bool> FxAsync.some(FutureOr<bool> Function(T a) f) // chain

Lecture

some is every's mirror image: it scans left to right and short-circuits the moment it finds a match, returning true right away. If it never finds one — including on an empty iterable, which is the opposite vacuous case from every — it returns false only after checking everything.

Unlike every, Fx does define its own .some() chain method (it isn't already covered by Iterable, since Dart's own Iterable has no built-in some — the closest thing, any, does the same job under a different name).
