map

Devuelve un iterable perezoso con los valores que resultan de pasar cada elemento por una función.

Iterable<B> map<A, B>(B Function(A a) f, Iterable<A> iterable) FxAsyncIterable<B> mapAsync<A, B>(FutureOr<B> Function(A a) f, FxAsyncIterable<A> iterable) Fx<R> Fx.map<R>(R Function(T a) f) // chain FxAsync<R> FxAsync.map<R>(FutureOr<R> Function(T a) f)

Lección

map es el transformador más fundamental: aplica una función a cada elemento. En FxDart es perezoso — llamar a map no hace absolutamente nada de trabajo. La función solo se ejecuta cuando un operador terminal (toList, each, reduce, …) tira de los valores a través del pipeline. Eso significa que puedes hacer map sobre una secuencia enorme — incluso infinita — mientras solo pidas lo que necesitas.

Viene en forma data-first (map(f, iterable)) y como método de cadena (fx(iterable).map(f)). Ambas devuelven el mismo resultado perezoso.
